### 设计文档

#### 项目介绍：

本项目使用react框架、nodejs搭建，实现了基本的抽奖功能，其中由node端完成抽奖概率计算，前端获取后端接口进行奖品初始化、获取中奖奖项

#### 仓库使用说明

##### 启动后端：

分支：Lottery-backend

`npm install` 

`node app.js` 

##### 启动抽奖页面：

分支：Lottery-frontend

`npm install` 

`npm start`
